### Глоссарий современных технологий и продуктов для DevOps

- **AirFlow (Apache)** - платформа для оркестрации рабочих процессов на Python.

- **Airbyte (Airbyte)** - платформа с открытым исходным кодом для интеграции данных (ELT).

- **Amazon SageMaker (Amazon Web Services)** - управляемая облачная платформа для создания и развертывания моделей ML.

- **Ansible (Red Hat)** - инструмент для автоматизации конфигурации и развертывания.

- **Argo CD (Argo Project)** - инструмент GitOps для непрерывной доставки в Kubernetes.

- **Argo Workflows (Argo Project)** - движок для оркестрации рабочих процессов в Kubernetes.

- **AWS (Amazon Web Services)** - публичная облачная платформа от Amazon.

- **Azure (Microsoft)** - публичная облачная платформа от Microsoft.

- **Backstage (Linux Foundation)** - фреймворк для создания порталов разработчиков (IDP).

- **Bamboo (Atlassian)** - сервер CI/CD от Atlassian.

- **BigQuery (Google)** - бессерверное, масштабируемое облачное хранилище данных.

- **ClickHouse (ClickHouse, Inc.)** - высокопроизводительная колоночная СУБД.

- **Consul (HashiCorp)** - инструмент для обнаружения сервисов и конфигурации сетей.

- **containerd (Cloud Native Computing Foundation)** - высокоуровневая среда выполнения контейнеров.

- **Crossplane (Cloud Native Computing Foundation)** - фреймворк для управления облачной инфраструктурой через Kubernetes.

- **Databricks (Databricks)** - унифицированная платформа для данных и ИИ на основе Spark.

- **Docker (Docker, Inc.)** - платформа для разработки, доставки и запуска контейнеров.

- **Docker Compose (Docker, Inc.)** - инструмент для определения и запуска многоконтейнерных приложений.

- **Docker Swarm (Docker, Inc.)** - встроенный в Docker инструмент для оркестрации контейнеров.

- **DVC (Iterative)** - система контроля версий для данных и моделей машинного обучения.

- **Envoy (Cloud Native Computing Foundation)** - высокопроизводительный прокси-сервер для сервисных сетей.

- **Estuary (Estuary)** - платформа для интеграции данных в реальном времени.

- **Flink (Apache)** - фреймворк для обработки потоковых и пакетных данных.

- **FluxCD (Flux project)** - инструмент GitOps для непрерывной доставки в Kubernetes.

- **Git (Software Freedom Conservancy)** - распределенная система контроля версий.

- **GitHub (Microsoft)** - веб-платформа для хостинга и совместной разработки Git-репозиториев.

- **GitLab (GitLab)** - единая платформа для полного жизненного цикла разработки ПО.

- **Grafana (Grafana Labs)** - платформа для аналитики, мониторинга и визуализации.

- **Helm (Cloud Native Computing Foundation)** - менеджер пакетов для Kubernetes.

- **HyperDX (HyperDX)** - платформа для наблюдаемости с открытым исходным кодом.

- **Istio (Google, IBM, Lyft)** - сервисная сеть (service mesh) для микросервисов.

- **Jenkins (Jenkins project)** - автоматизированный сервер с открытым исходным кодом для CI/CD.

- **Jira (Atlassian)** - система для управления проектами и отслеживания задач.

- **Kafka (Apache)** - распределенная потоковая платформа для обработки событий.

- **KFServing / KServe (Kubeflow project)** - высокопроизводительный сервер для развертывания моделей ML в Kubernetes.

- **Kubernetes (Cloud Native Computing Foundation)** - платформа для оркестрации контейнеров.

- **K3s (Rancher)** - легковесная версия Kubernetes для edge-вычислений.

- **Kubeflow (Kubeflow project)** - платформа для машинного обучения в Kubernetes.

- **Kustomize (Kubernetes SIGs)** - инструмент для настройки конфигураций Kubernetes.

- **Loki (Grafana Labs)** - система для агрегации и хранения логов.

- **MicroK8s (Canonical)** - легковесная, автономная версия Kubernetes.

- **MLflow (Databricks)** - платформа для управления жизненным циклом машинного обучения.

- **NGINX (F5 Networks)** - веб-сервер, обратный прокси и балансировщик нагрузки.

- **Nix/NixOS (NixOS Foundation)** - система управления пакетами и операционная система.

- **OpenShift (Red Hat)** - платформа для контейнерных приложений на основе Kubernetes.

- **OpenTelemetry (Cloud Native Computing Foundation)** - стандарт для генерации и сбора телеметрии.

- **OpenTofu (Linux Foundation)** - инструмент Infrastructure as Code, форк Terraform.

- **Pachyderm (Pachyderm)** - платформа версионирования данных и конвейеров для воспроизводимого ML.

- **Podman (Red Hat)** - инструмент для управления контейнерами без демона.

- **Prefect (Prefect Technologies)** - платформа для оркестрации рабочих процессов данных и ML.

- **Prometheus (Cloud Native Computing Foundation)** - система мониторинга и оповещения.

- **Pulumi (Pulumi)** - инструмент Infrastructure as Code с поддержкой языков общего назначения.

- **Rancher (SUSE)** - платформа для управления несколькими кластерами Kubernetes.

- **Seldon Core (Seldon)** - платформа для развертывания и мониторинга моделей ML в Kubernetes.

- **Slack (Salesforce)** - платформа для командной коммуникации.

- **Snowflake (Snowflake Inc.)** - облачное хранилище данных с разделением вычислительных ресурсов и хранения.

- **Spark (PySpark) (Apache)** - распределенный движок для обработки больших данных.

- **Tableau (Salesforce)** - инструмент для бизнес-аналитики и визуализации данных.

- **Terraform (HashiCorp)** - инструмент Infrastructure as Code для безопасного построения облачной инфраструктуры.

- **Tekton (Cloud Native Computing Foundation)** - платформа для создания CI/CD-систем в Kubernetes.

- **Vault (HashiCorp)** - инструмент для безопасного хранения и управления секретами.

- **Visual Studio Code (Microsoft)** - редактор исходного кода с поддержкой расширений.

- **ZITADEL (ZITADEL)** - платформа управления идентификацией и доступом с открытым исходным кодом.
